========== 실행 흐름 예시 ==========

1. 사용자가 URL 입력
         ↓
2. 브라우저 엔진이 네트워킹 계층에 요청
         ↓
3. 네트워킹 계층이 서버에서 HTML/CSS/JS 다운로드
         ↓
4. 렌더링 엔진이 HTML/CSS 파싱 (DOM/CSSOM 생성)
         ↓
5. JavaScript 엔진이 JS 코드 실행 (DOM 조작 가능)
         ↓
6. 렌더 트리 구축 → 레이아웃 → 페인팅 → 합성
         ↓
7. UI 백엔드가 폼 컨트롤 등 네이티브 위젯 렌더링
         ↓
8. 화면에 최종 결과 표시
